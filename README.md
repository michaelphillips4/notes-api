# Elysia with Bun runtime

## Getting Started

```bash
curl -fsSL https://bun.sh/install | bash
bun create elysia notes-api //already done
bun install
bun run
```

## Development
To start the development server run:
```bash
bun run dev
```

Open http://localhost:3000/ with your browser to see the result.
